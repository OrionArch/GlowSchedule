# mockups

<!-- Parent: ../AGENTS.md -->

Design mockups and icon generation utilities for GlowSchedule.

## Key Files

| File | Purpose |
|------|---------|
| `design_preview.html` | HTML-based visual design preview of the app UI |
| `generate_icons.py` | Python script for generating app icons from source assets |
| `stitch_cat.png` | Source image asset used for icon generation |

## Usage

```bash
# Generate icons (requires Python with Pillow)
python generate_icons.py
```

Open `design_preview.html` in a browser to view the UI mockup.
